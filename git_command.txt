This is git Command example file.

1. [add] Command
    =>  git add [file_name]
    ex) git add git_command.txt

2. [stage] Command
    => git stage [file_name]
    ex) git stage ddi_top.v

3. [commit] Command
    => git commit -m "log message"
    ex) git commit -m "Add file git_command.txt"

4. [reset] Command
    => git reset HEAD <file>
    ex) git reset HEAD git_command.txt
